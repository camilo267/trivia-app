# Trivia-app

Trivia App for testing your knowledge about technolofy

#### Technologies 

Vuejs, Vuetify and [OpenTriviaDataBase](https://opentdb.com/) API

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

```
npm run build
```
